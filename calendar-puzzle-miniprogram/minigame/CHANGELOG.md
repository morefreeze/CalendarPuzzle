# Changelog

## [0.2.2] — 2026-05-14

### 新增

- **多基底解 pack 接入（免费版）**：把按 `(月, 日, 星期)` 索引的完整解 pack（每键 20 个不同的完整解，gzip 后 ~690 KB）打进主包。每次换题不仅块的组合不同，**整张棋盘的底盘布局**也可能不同，玩家不会再看到熟悉的"骨架"。
- 单日可玩题量大幅提升：黑铁 ~50 → **727**，白银 ~100 → ~2000，黄金 ~30 → ~600，钻石 ~10 → **199**（以 2026-05-14 为例）。

### 修复

- **手动选题面板滚动误选**：在选题面板里上下滚动浏览缩略图、抬手时若落在某个缩略图上，会被误判为"点击选题"并弹出体力消耗确认。现已区分滚动与点击：单次移动距离 ≥ 8px 即标记为滚动，抬手时不再触发选题。

### 工程

- 新增 `gen_pack.py`：按 `(月, 日, 星期)` 离线生成解 pack，支持 `--top-k`（免费版）/ `--all`（付费版）/ `--parallel` 多核加速。
- pack 按 56 字符棋盘字符串紧凑编码，自动产出 `.json` + `.json.gz` 两份；payload 与 miniprogram `solvedBoard` 格式直接兼容。

## [0.2.1] — 2026-05-14

### 新增

- **持久化通关记录**：按「日期 + 难度」分桶记录已通关的 combo 索引（`wx.setStorageSync`，key = `calendarPuzzleWonCombos`）。冷启动、返回主菜单后重进、好友分享链接打开同一谜题，均能看到已完成的题目。
- **手动选题面板「已通关」勾选**：在 `选择题目` 缩略图右上角显示绿底白勾角标，对应当前 `(日期, 难度)` 桶下所有已通关的 combo；与"当前题目"的绿色高亮边框可同时共存。
- **拖动落点预览**：拖动方块越过棋盘时，按 `Math.round` 吸附位置在棋盘上叠加一份半透明落点预览（alpha 0.35），与实际落点完全一致。
  - 可放置：方块原色。
  - 不可放置（越界 / 重叠 / 碰到日期格）：浅灰 `#bbbbbb`。
  - 跟手的悬浮 ghost 仍跟随手指、不吸附，作为操作反馈。

### 改动

- **去除冗余 toast**：移除"放置成功！"和"已移除方块"两条提示——棋盘状态已经自表达。错误/警告类（无法放置、越界、体力不足、方向锁定、提示）保留。
- **修复换题状态丢失（latent bug）**：`executeRandomSwitch` / `executeManualSwitch` 在调用 `onSwitchPuzzle` 时透传 `_playedCombos` / `_wonCombos`，让随机换题"避免重复"的逻辑真正跨换题生效（与提交 `858b4df` 的意图对齐）。

## [0.2.0] — 2026-05-13

### 新增

- **邀请朋友挑战**：通关后画面中部出现"🎯 邀请朋友挑战这一题"按钮，一键弹出微信好友选择面板。分享卡片标题随当前难度变化（例如"日历方块「黑铁」挑战 — 来比比谁快！"）。
- **可复现谜题分享链接**：分享卡片的 `query` 携带 `d`（难度）/`c`（combo 索引）/`date`（题目日期），朋友点开冷启时**直接进入同一道题**，跳过难度选择界面，且**不消耗体力**。

### 改动

- **棋盘配色低对比化**：月/日/周三类格子改为接近的绿色渐变（`#E8F5E9` / `#C8E6C9` / `#A5D6A7`），整体更柔和。**今日标记（uncoverable）改为深橙 `#FF7043`** —— 在绿底上对比强烈，一眼可辨。
- **主题接口预留**：`board.js` 中新增 `THEMES` 调色板表 + `setCellTheme(name)` API，未来新增主题只需添一项后调用 `B.setCellTheme('xxx')`。
- **预览区高度固定**：选中方块后的预览行固定为 **4 格高**，方块旋转 / 翻转时整行不再变高，视觉稳定。
- **去除"已选择: X"提示文字**：预览区已显示形状本身，提示文字属于冗余。
- **返回按钮重设计**：原"返回"按钮从控制行底部移到屏幕**左上角**，作为左向箭头（`<`）展示。
- **左滑返回手势**：在屏幕左侧 24px 内起手、向右滑动 >60px（横向位移 > 纵向位移）即可返回难度选择界面。
- **旋转 / 翻转重新布局**：从控制行移到所选方块**左右两侧**，比例 `[翻转 0.172] [空 0.05] [方块 0.556] [空 0.05] [旋转 0.172]`。
- **"已选择 X"提示移除**：方块已经显示在预览区，不再需要文字说明。
- **「换题」改为 split-button**：
  - 主体显示当前换题模式（"随机换题"紫色 / "手动换题"青色）+ 当前题目总数。
  - 右侧 `▾` 箭头一键切换 random / manual 模式（无下拉弹层）。
  - 移除独立的"随机/手动"模式切换按钮。
- **进行中换题需消耗体力**：未通关时点"换题"会先弹 `wx.showModal` 二次确认，确认后扣除该难度对应体力（黑铁 3 / 白银 5 / 黄金 7 / 钻石 9）。通关后换题免费、免确认。
- **体力不足直接提示**：进行中换题时若体力不足，直接显示"体力不足！需要 X 点，当前 Y 点"，不进入确认弹层。
