# 将 Web 前端移植到微信小程序

## 概述
将 `my-cal` 中的完整游戏功能移植到 `calendar-puzzle-miniprogram`，包括拖拽交互、游戏逻辑、状态管理、API 集成等。

## 关键差异分析

### Web 版本（my-cal）功能
- ✅ 完整的拖拽系统（react-dnd + html5-backend）
- ✅ 计时器和游戏ID管理
- ✅ 求解器 API 集成
- ✅ 胜利检测和游戏状态持久化
- ✅ 键盘快捷键支持
- ✅ 双击返回面板、长按翻转

### 小程序版本现状
- ⚠️ 基础拖拽框架（react-dnd，但不兼容小程序）
- ❌ 缺少计时器
- ❌ 缺少求解器集成
- ❌ 缺少胜利检测
- ❌ 缺少状态持久化

## 技术挑战

1. **拖拽库替换**：react-dnd 在小程序中不工作，需使用小程序原生拖拽 API
2. **状态持久化**：localStorage 需替换为 wx.setStorageSync
3. **API 请求**：fetch 需替换为 wx.request
4. **键盘事件**：小程序不支持键盘，需设计触摸交互替代方案
