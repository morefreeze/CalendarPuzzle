## 前后端交互重构提案

**核心问题**：`api.tsx` 中硬编码了 `http://localhost:5001/api`，导致无法在生产环境使用。

**推荐方案**：云端部署后端 + 环境配置系统

**主要变更**：
1. **创建环境配置层**：`config/api.ts` 根据环境变量动态选择 API 地址
2. **重构 API 通信层**：添加超时控制、重试机制、错误处理
3. **部署后端服务**：选择 Railway/Fly.io/Heroku 之一部署 Python 后端
4. **配置域名白名单**：在小程序后台添加 API 域名

**实施步骤**：
- 阶段一（配置层）：创建环境配置、重构 API 层 → 约 10h
- 阶段二（部署）：部署后端服务、配置域名 → 约 6h
- 阶段三（集成）：更新小程序配置、测试 → 约 3h
